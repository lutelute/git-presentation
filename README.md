## キーボード操作

- `→` / `←` : スライド移動
- `g` : 最初のスライドへ
- `G` : 最後のスライドへ
- 各スライドの ▶ ボタンでターミナルデモが再生されます

- 自己完結型HTML（CDN依存なし、GitHub Pages対応）
- ターミナルアニメーションエンジン内蔵
